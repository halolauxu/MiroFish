# 2026-03-20 策略宇宙研究：赚钱执行蓝图

## 1. 这项研究最终要回答什么

这项研究的最终目标，不是证明“模型很聪明”，也不是证明“图谱能传播”，而是回答一个更直接的问题：

> 在当前 A 股可交易约束下，什么样的事件信号，能稳定地转成真钱收益？

因此，最终研究结论必须同时包含两层：

1. **研究层结论**
   - Walk-Forward Sharpe
   - 单信号年化收益代理值
   - Alpha 来自哪些桶、哪些方向、哪些链路

2. **赚钱层结论**
   - 这个 alpha 应该如何执行
   - 是做多、做空、对冲、相对收益，还是“只用于规避”
   - 哪些信号只是研究上有效，但不能直接变现

当前阶段最重要的，不是立刻把所有东西做成组合 NAV，而是先把“赚钱路径”判断清楚。

## 2. 回溯整个研究后的主结论

### 2.1 已经被证伪的东西

早期更宽口径的图谱/事件研究，并没有证明“广谱图谱传播 = 可稳定赚钱”。

来自历史产物的证据：

- `full_validation_20260319_0712.md`
  - 全量 Sharpe 只有 `0.44`
  - WF-OOS Sharpe 只有 `0.04`
  - 图谱优于随机：`FAIL`
  - 未反应优于已反应：`FAIL`

这说明两件事：

1. **“广撒网的下游传播”不是当前可直接下注的主 alpha**
2. **“未反应信息差”并不是这套系统里最稳定的赚钱来源**

### 2.2 已经被验证出来的东西

最新严格样本下，真正跑出来的 edge 来自更窄、更偏执行的部分。

来自最新 refreshed 结果：

- `shock_wf_backtest_20260320_local_refresh.md`
  - 5D 全量 Sharpe：`1.24`
  - 5D OOS Sharpe：`2.50`
  - 5D 全量调整收益：`+0.0113`
  - 5D OOS 调整收益：`+0.0188`

- `shock_bucket_study_20260320_local_refresh.md`
  - `hop=0`：`+0.0206`
  - `hop>0`：`-0.0034`
  - `policy_risk`：`+0.0419`
  - `cooperation`：`-0.0164`
  - `buyback`：`-0.0060`
  - `management_change`：`-0.0060`

这说明：

1. **当前 alpha 主要不在“下游慢反应”**
2. **当前 alpha 主要在“源头事件方向延续”**
3. **当前最强 alpha 主要在负向/规避侧，而不是纯做多侧**

### 2.3 当前 alpha 到底在哪里

用一句话概括：

> 当前最强的 alpha，不是“图谱下游信息差 alpha”，而是“源头事件的方向延续 alpha”，尤其是 `policy_risk` 这类负向事件的 `avoid` alpha。

进一步拆开看：

- **最强位置**
  - `policy_risk`
  - `hop=0` 源头事件
  - 已反应后的继续走弱/继续走趋势

- **次强位置**
  - 部分 `ma`，但必须按 source 再拆，不能当通用桶

- **当前不该下注的位置**
  - `cooperation`
  - `buyback`
  - `management_change`
  - 泛化的 `hop>0`

### 2.4 方向侧的关键事实

当前研究不是“纯做多 alpha”。

在 refreshed 样本里：

- `long`
  - 5D 调整收益：`-0.0142`
  - Sharpe：`-1.88`

- `avoid`
  - 5D 调整收益：`+0.0270`
  - Sharpe：`2.92`

所以当前研究的本质，更接近：

- **方向识别 alpha**
- **风险规避 alpha**
- **负向事件 continuation alpha**

而不是“给我一组多头股票，我长期持有就能赚钱”。

## 3. 最终怎么写 WF Sharpe 和年化收益

最终研究结论里，必须把“年化收益”分成两个口径，不然会把研究结论写歪。

### 3.1 口径 A：研究代理年化收益

这是一种**信号强度代理值**，用于衡量 alpha 强弱，不是最终可交易 NAV。

计算方法：

- 用 `5D 平均方向调整收益`
- 按 `252 / 5` 做年化

最新 refreshed 样本：

- 全量：
  - 5D 调整收益：`+1.13%`
  - 简单年化代理：`+56.9%`
  - 复利年化代理：`+76.2%`

- OOS：
  - 5D 调整收益：`+1.88%`
  - 简单年化代理：`+94.8%`
  - 复利年化代理：`+155.7%`
  - 但样本只有 `28` 条，不能外推成稳定收益承诺

这个口径的意义是：

- 用来比较不同桶
- 用来比较不同版本研究
- 用来判断 alpha 强弱

不是用来回答“账户能赚多少”

### 3.2 口径 B：执行年化收益

这才是最后真正跟“赚钱”有关的口径。

它必须依赖：

- 资金分配
- 信号重叠
- 仓位上限
- 交易可行性
- `avoid` 如何货币化
- 对冲工具
- 实际滑点/成交约束

当前阶段，这个口径**还没有严谨产出**，不能拿信号代理值冒充。

所以最终结论写法应该是：

> 当前严格 WF 已验证存在正向 alpha，研究代理年化收益为 X%；但该 alpha 主要位于 `avoid / negative-event continuation` 侧，真实执行年化收益仍取决于后续的变现路径设计。

## 4. “赚钱”不是一个组合，而是三层执行路径

如果目标是赚钱，那当前最合理的执行蓝图，不是直接说“做一个纯多组合”，而是把研究结果映射成三层可变现路径。

### 路径 A：风险规避 / 减仓层

这是当前最成熟、最可信的一层。

对应信号：

- `policy_risk`
- 强负向源头事件
- 明确 `avoid` 且已反应后仍有延续性的桶

变现方式：

- 已有持仓的减仓/清仓规则
- 新仓过滤
- 风险预算收缩
- 作为多头组合的 overlay

这层的特点是：

- 最容易落地
- 不依赖融券
- 对真钱最有价值

### 路径 B：负向事件 alpha 的对冲化变现

如果目标不只是规避，而是把 `avoid` 真正变成收益，那么就要把当前 strongest alpha 转成可交易结构。

可行形式不是“口头上的做空”，而是以下几种之一：

- 融券可做时，直接 short
- 做行业/指数对冲后的相对收益
- 做事件 loser basket vs hedge basket
- 作为现有组合的 alpha overlay，而不是独立 long-only 策略

这条路径是当前研究最接近“真钱 alpha”的主干。

### 路径 C：正向事件做多层

这层目前还没有被证明足够强。

当前 `long` 侧整体是负的，所以不能把“所有正向事件”直接当成多头策略。

只有在以下条件同时满足时，正向做多才值得推进：

- 桶本身通过严格 WF
- OOS 样本够
- 不是只靠单一 source 偶然拉动
- 可以独立于 `avoid` 侧成立

当前最接近候选的是：

- 经过 source 拆分后的部分 `ma`

但它还不该被定义成主赚钱路径。

## 5. 下一阶段研究，不是“做组合”，而是“做变现映射”

当前最值钱的下一步，不是泛泛地做一个完整组合，而是做下面四件事：

### 5.1 先把研究对象拆成两套系统

不要再把它们混在一起：

- 系统 A：`source-event continuation`
- 系统 B：`downstream propagation`

当前看，A 有真钱潜力，B 还在验证期。

### 5.2 把 `avoid` 从“标签”升级成“变现方案”

这是最关键的一步。

如果 `avoid` 只是“别买”，那它只是风控；
如果 `avoid` 能映射成：

- 减仓规则
- 对冲规则
- relative short / hedge 规则

它才是真 alpha。

### 5.3 把最终研究结论模板固定下来

以后每版结论都统一输出：

1. WF Sharpe
2. 研究代理年化收益
3. Alpha 所在桶
4. Alpha 所在方向（long / avoid）
5. 变现路径（减仓 / 对冲 / 相对收益 / 观察）

### 5.4 只对“能变现”的桶继续投入

继续研究的优先级应该是：

1. `policy_risk`
2. `hop=0` 源头事件
3. 可拆 source 的 `ma`
4. `avoid` 的实际执行映射

而不是继续平均投入到所有 event type。

## 6. 一句话蓝图

这项研究当前真正通向“赚钱”的蓝图，不是：

> 用图谱找一堆股票，然后做一个 A 股纯做多组合。

而是：

> 先用严格数据底座和 WF 找出真正有方向 alpha 的事件桶，确认 alpha 主要在 `源头事件 + 负向 continuation + avoid`；再把 `avoid` 从研究标签，升级成减仓、对冲、相对收益的执行规则；最后只对那些能稳定变现的桶做资本化。

也就是说，当前最应该推进的，不是“组合化”，而是：

**把 alpha 和变现路径一一对上。**
